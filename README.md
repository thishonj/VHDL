VHDL Learning Repository

This repository contains my hands-on learning and implementation of VHDL concepts based on the tutorials from:
🔗 https://vhdlwhiz.com/basic-vhdl-tutorials/

🚀 About This Repository
I created this repository to:
Practice and strengthen my understanding of VHDL
Implement concepts through code examples
Help others who are getting started with digital design and FPGA development

🧠 Topics Covered
VHDL Basics
Signals vs Variables
Processes and Sequential Logic
Combinational Logic Design
Finite State Machines (FSM)
Testbenches and Simulation
Timing Concepts

🛠️ Tools Used
VHDL (IEEE Standard)
Simulation Tools Questa Altera Starter FPGA Edition, ModelSim

🎯 Who Is This For?
Beginners learning VHDL
Students in Electronics / Embedded Systems
Anyone interested in FPGA or ASIC design

💡 Learning Approach
Instead of just reading tutorials, I:
Implemented each concept in code
Tested using simulation
Structured examples for easy understanding

🤝 Contributions
This is a personal learning repository, but suggestions and improvements are always welcome!
