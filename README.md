VHDL Learning Repository

This repository contains my hands-on learning and implementation of VHDL concepts based on the tutorials from:

🔗 https://vhdlwhiz.com/basic-vhdl-tutorials/

🚀 About This Repository

I created this repository to:

Practice and strengthen my understanding of VHDL
Implement concepts through code examples
Help others who are getting started with digital design and FPGA development

Each folder/file corresponds to specific concepts covered in the tutorials.

📂 Repository Structure
├── 01_basic_syntax/
├── 02_signals_and_variables/
├── 03_process_blocks/
├── 04_finite_state_machines/
├── 05_testbenches/
├── ...

⚠️ Folder names may vary based on the topics covered.

🧠 Topics Covered
VHDL Basics
Signals vs Variables
Processes and Sequential Logic
Combinational Logic Design
Finite State Machines (FSM)
Testbenches and Simulation
Timing Concepts
🛠️ Tools Used
VHDL (IEEE Standard)
Simulation Tools (e.g., ModelSim / GHDL / Vivado Simulator)
FPGA Tools (optional, depending on implementation)
